# Feature List - TaskSphere

## Auth
- Register / Login with JWT
- User profile page

## Boards & Tasks
- Create/edit/delete boards
- Add lists to boards
- Add tasks to lists
- Task details (title, description, due date, priority, tags)

## Team Collaboration
- Create teams
- Invite users
- Assign roles: admin, editor, viewer
- View activity log

## UI/UX
- Clean dashboard
- Dark mode
- Toast notifications
- Responsive design (desktop & mobile)

## Admin
- View all users and teams
- Block user / delete team
- Dashboard analytics

## Extra
- Search & filter tasks
- Comments per task (optional)
- File attachments (optional)

## Deployment
- AWS EC2 + Gunicorn + Nginx
